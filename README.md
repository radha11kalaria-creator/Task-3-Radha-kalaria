# FreshBasket - Organic Food Delivery App

## Overview

FreshBasket is a premium organic food delivery mobile application designed to provide users with a seamless and enjoyable shopping experience. The project focuses on creating a clean, modern, and trustworthy interface that promotes healthy living and convenient online grocery shopping.

## Project Goals

* Create a strong visual identity
* Improve user experience
* Design a modern mobile interface
* Promote trust, freshness, and convenience

## Screens Included

* Splash Screen
* Onboarding Screens
* Login & Sign Up
* Product List

* Cart Screen
* Order Success Screen
